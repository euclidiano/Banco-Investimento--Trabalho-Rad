# Banco-Investimento--Trabalho-Rad
Este projeto é um protótipo de aplicativo bancário desenvolvido para demonstrar a um cliente como funcionaria o cadastro de usuários e o direcionamento com base no saldo da conta.

- Se o usuário tiver menos de R$ 15.000, ele é levado para uma tela com informações e sugestões de investimentos para o futuro.
- Se o usuário tiver R$ 15.000 ou mais, ele é direcionado para uma tela de investimentos.
